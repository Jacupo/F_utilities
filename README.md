# F_utilities

## Installation
Enter the Pkg REPL by pressing `]` from the Julia REPL.
```
pkg> add  https://github.com/JuliaMatrices/SpecialMatrices.jl
pkg> add https://github.com/Jacupo/F_utilities
```
